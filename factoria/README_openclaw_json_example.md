# Uso de `openclaw.json.example`

Archivo JSON de referencia **sin secretos**. Copialo como `openclaw.json` en el volumen de datos de OpenClaw en el servidor y reemplazá:

- `REEMPLAZAR_TELEGRAM_BOT_TOKEN` — token del bot Telegram  
- `REEMPLAZAR_GATEWAY_TOKEN` — token de la Control UI  

Completá `agents.list` con el resto de perfiles si usás los 10 agentes (ver tabla en el [README.md](../README.md) del repo).
